# Defaults & Fallbacks

Autoclean uses sensible defaults so you can type less.

## Selection Priority

- Task: CLI argument → active Task → prompt
- File: CLI argument → task config → active Source → prompt
- Output: CLI argument → `Workspace/output/` → current directory

Keep this model in mind and you’ll move fast without surprises.
